# Check Workday Status

Run `wd status` to see:
- Current timer state (focus/break/paused)
- Time remaining
- Pomodoros completed vs planned
- Tasks completed
- Break counts

```bash
wd status
```

Also available: `wd stats` for overall statistics, `wd history` for past days.
